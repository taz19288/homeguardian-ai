# HomeGuardian AI 🏠

AI-powered home maintenance assistant that helps homeowners protect, maintain, and increase the value of their home.

## Vision
Become the AI operating system for homeownership.

## Problem
Homeowners struggle to:
- Track maintenance tasks
- Understand home issues
- Manage warranties and documents
- Know when repairs are needed

## Solution
HomeGuardian AI provides:
- AI home assistant
- Maintenance tracking
- Home health insights
- Document storage
- Personalized recommendations

## Status
MVP in development (Week 1)

## Tech Stack
- Flutter (mobile app)
- Supabase (backend)
- OpenAI API (AI layer)
